Welcome to ShellCraft.
======================

ShellCraft is a hackable game about mining, crafting, and puzzling, loosely based on Ted Chiang's short story `"Seventy-Two Letters"`_.

To download and play the game right now, simply run:

.. code-block:: console

    # Install ShellCraft
    $ pip install shellcraft

    # Run ShellCraft:
    $ shellcraft
