ShellCraft is a game about mining, crafting, and puzzling, loosely based on Ted Chiang's short story `"Seventy-Two Letters"`_.

.. code-block:: console

    # Install ShellCraft
    $ pip install shellcraft

    # Run ShellCraft:
    $ shellcraft


Read the full documentation `here`_.
