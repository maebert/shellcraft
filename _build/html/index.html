ShellCraft is a hackable game about mining, crafting, and puzzling, loosely based on Ted Chiang’s short story “Seventy-Two Letters”.

To download and play the game right now, simply run:

# Install ShellCraft
$ pip install shellcraft

# Run ShellCraft:
$ shellcraft
